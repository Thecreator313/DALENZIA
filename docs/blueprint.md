# **App Name**: Fest Central

## Core Features:

- Admin Dashboard: Admin Dashboard: Provides a centralized overview with key metrics, quick access to manage teams, categories, programs, stage control, and user settings.
- Team Dashboard: Team Dashboard: Offers a summary of team activities, student assignments, and quick links to add/manage students and generate reports.
- Judge Dashboard: Judge Dashboard: Displays a summary of judging assignments, quick access to input judging points, and profile management.
- Stage Control Dashboard: Stage Control Dashboard: Provides a summary of upcoming programs and essential stage information for real-time management.
- Public Page: Public Page: Displays general information about the festival, including event details, schedule, and sponsors.
- AI Category Tool: Category Suggestions: An AI tool provides smart suggestions for categories based on entry details, ensuring relevant categorization.

## Style Guidelines:

- Primary color: Light Blue (#ADD8E6), evoking a sense of calm and clarity, aligning with the need for accessible and digestible information on each dashboard.
- Background color: Very light gray (#F0F0F0), to ensure a clean and modern feel while supporting readability and focus.
- Accent color: Soft Purple (#E0B0FF), to provide a gentle contrast and highlight interactive elements or key data points without overwhelming the user.
- Body and headline font: 'Inter', a grotesque-style sans-serif providing a modern and neutral look for readability across dashboards and the public page.
- Code font: 'Source Code Pro', monospaced sans-serif used when displaying the Tailwind CSS code snippets for UI customization.
- Consistent layout across all panels, using Tailwind CSS grid and flexbox for responsive design.
- Simple, clear icons to represent menu items and actions, enhancing usability.
